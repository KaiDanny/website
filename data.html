Data Deletion Instructions

Effective Date: January 1, 2026

This page explains how users can request deletion of their game-related data.

What Data We Store

Our game may store limited gameplay-related information such as:

  • Game progress
  • Leaderboard scores
  • Cloud save data
  • Player preferences

We do not collect sensitive personal information such as passwords, payment details, or physical addresses.

How to Request Data Deletion

If you would like your game-related data to be deleted, please contact us using the email below.

tenfold@szsmbkj.com

Please include the following information in your request:

  • Your Facebook Instant Game Player ID or App-Scoped User ID
  • The name of the game
  • A brief request for data deletion

Deletion Process

After receiving a valid request, we will:

  • Verify the request
  • Delete associated gameplay data from our systems
  • Complete the deletion within a reasonable timeframe

Facebook Data Deletion Callback

If you initiated the request through Facebook, Facebook may send us a data deletion callback request automatically.

Contact

If you have any questions regarding this Data Deletion Policy, please contact:
